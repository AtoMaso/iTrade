This application is EST social network with a dash of EST directory where EST members can share any information they can think of. Anounce anything they think needs announcing, share thier software development experiences or any other expirience. Share thier professional or personal live knowledge, but in the same time they can also find all information they need about EST. The EST member will need to become a member of the web site first.
This web application was built using the following software development technologies:
  • Angular 2 framework
  • Typescript programming language.
  • Bootstrap CSS framework
  • NodeJS framework
  • SystemJS framework
