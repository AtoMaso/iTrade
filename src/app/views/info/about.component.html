This application is small trading application where anyone can trade their unwanted things for a wanted things. To be able to trade for any product freely you will need to become a trader. Registring to the site is free with no membership fees.
This web application was built using the following software development technologies:
  • Angular framework version 5.0
  • Typescript programming language version 2.4.2
  • Bootstrap CSS framework version 3.3.7
  • NodeJS framework version 8.9.3
  • SystemJS framework version 2.4.1
